Grunt Tasks
===========

Make sure that nodejs_ is installed. Then in the project root run ::
	
	$ npm install
	or
	$ make node_modules

.. _nodejs: http://nodejs.org/download/

--------------------------------------

To use ``browser-sync`` for Time-saving synchronised browser testing::

	$ grunt browser-sync
